1. Prompt para Consertar e Profissionalizar o PDF
Este prompt foca em resolver o erro NaN e trazer o design visual.

Copie e cole isso para sua IA:

"Estou desenvolvendo um dashboard financeiro em React. A função de exportação atual está muito ruim e quebrada (mostrando 'NaN' nos valores).

Preciso que você reescreva a função handleExportPDF. Quero um Relatório Gerencial Executivo profissional.

Requisitos Técnicos:

Bibliotecas: Use html2canvas para capturar os gráficos (charts) como imagens e jspdf para montar o documento A4.

Correção de Dados (NaN): A função atual não está lendo os estados metrics, cashFlow e dre corretamente. Certifique-se de passar esses objetos como argumentos para a função de geração ou acessar o estado global corretamente para que os valores apareçam (ex: R$ 20.039,64) em vez de 'NaN'.

Estrutura do Relatório (Layout A4):

Cabeçalho: Coloque um título 'Relatório Financeiro Analítico', a data de geração e o período (ex: 'Este Mês').

Seção 1 - Sumário Executivo: Renderize o texto gerado pela IA (aquele insight sobre saúde financeira sólida) em destaque, com um fundo cinza claro arredondado.

Seção 2 - KPIs: Crie uma linha com os 4 cards principais (Receita, Lucro, A Receber, A Pagar) formatados como 'mini cards' no PDF.

Seção 3 - Gráficos: Capture o gráfico de 'Previsão de Fluxo de Caixa' e o de 'Tendência de Receita' e insira-os como imagens no PDF. Não tente recriar o gráfico com texto.

Seção 4 - DRE: Crie uma tabela nativa do PDF (autoTable plugin do jspdf é bom para isso) replicando a estrutura da DRE: Receita Bruta (verde), Custos (vermelho), Lucro (azul).

O resultado deve parecer um documento oficial da empresa, não uma página da web impressa."

2. Prompt para um Excel Útil (Multitabs)
O Excel atual parece uma lista simples de transações. Um relatório financeiro precisa de abas separadas.

Copie e cole isso para sua IA:

"Agora preciso melhorar a exportação para Excel (handleExportExcel). O atual é apenas uma lista plana de transações.

Quero gerar um arquivo .xlsx (use xlsx / SheetJS) que sirva como uma Planilha de Gestão Financeira. O arquivo deve ter 3 Abas (Worksheets) separadas:

Aba 1: Dashboard e Resumo

Coloque os totais no topo (Receita Total, Despesas, Lucro).

Liste o Fluxo de Caixa projetado (Mês 1, Mês 2, Mês 3) em colunas.

Aba 2: DRE Detalhada

Replique a estrutura da DRE (Demonstrativo de Resultados).

Colunas: Categoria | Valor | % do Total.

Separe visualmente Receitas, Custos Variáveis e Despesas Fixas.

Aba 3: Transações (Analítico)

A lista completa de transações (Data, Descrição, Valor, Categoria, Tipo).

Importante: Os valores monetários devem ser exportados como NÚMEROS (float) e não como texto (string), para que eu possa somar no Excel.

Aplique uma formatação básica de cabeçalho (negrito) nas planilhas."

Dica Extra: Por que está dando NaN?
Na imagem do PDF quebrado, o "Receita: R$ NaN" indica que a função que gera o PDF está tentando fazer contas (matemática) com variáveis que estão vazias (undefined) ou nulas no momento que você clica no botão. Ao usar o prompt acima, a IA vai forçar a verificação de que os dados (que aparecem corretamente na tela) sejam passados corretamente para o PDF.
